Module: azure-mobile-apps/express/tables

This module contains functionality for adding tables to an Azure Mobile App. It returns middleware that can be attached to an express app with some additional functions for registering tables.

(require("azure-mobile-apps/express/tables"))(configuration)

Create an instance of an express middleware function for routing and handling table requests.
Parameters:
Name Type Description
configuration configuration
Returns:
An express middleware function with additional members described below.

Methods

(inner) add(name, definition)

Register a single table with the specified definition.
Parameters:
Name Type Description
name string The name of the table. HTTP operations will be exposed on this route.
definition tableDefinition | express/tables/table The definition for the table.

(inner) import(path)

Import a file or folder of modules containing table definitions
Parameters:
Name Type Description
path string Path to a file or folder containing modules that export either a tableDefinition or table object. The path is relative to configuration.basePath that defaults to the location of your startup module. The table name will be derived from the physical file name.
