Module: azure-mobile-apps/data

Exposes data access operations for tables

(require("azure-mobile-apps/data"))(configuration)

Create an instance of the data provider specified in the configuration.
Parameters:
Name Type Description
configuration dataConfiguration The data provider configuration
Returns:
A function that accepts either a tableDefinition or table object and returns an object with the members described below.

Methods

(inner) read(query)

Execute a query against the table.
Parameters:
Name Type Description
query queryjs/Query The query to execute
Returns:
A promise that yields an array. The first element is the results of the query. If the includeTotalCount option was set on the query, the second element is the total number of records.

(inner) update(item)

Update a row in the table.
Parameters:
Name Type Description
item object The item to update
Returns:
A promise that yields an array. The first element is the number of records affected. The second element is the updated object.

(inner) insert(item)

Insert a row in the table.
Parameters:
Name Type Description
item object The item to insert
Returns:
A promise that yields the inserted object.

(inner) delete(id, version)

Delete an item from the table
Parameters:
Name Type Description
id string | number The id of the item to delete
version string Base64 encoded row version
Returns:
A promise that yields an array. The first element is the number of records affected. The second element is the deleted object.

(inner) undelete(id, version)

Undelete an item from the table
Parameters:
Name Type Description
id string | number The id of the item to delete
version string Base64 encoded row version
Returns:
A promise that yields an array. The first element is the number of records affected. The second element is the undeleted object.

(inner) truncate()

Clear all rows from the table
Returns:
A promise.
