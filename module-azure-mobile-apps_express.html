Module: azure-mobile-apps/express

This module is the entry point for adding an Azure Mobile App to an instance of an express web server. It is returned from the root azure-mobile-apps module when the configuration passed specifies the express platform.

(require("azure-mobile-apps/express"))(configuration)

Creates an instance of the azure-mobile-apps server object for express 4.x
